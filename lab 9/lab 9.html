LAB #9

INLINE CSS

Inline css is a way to set the style of a line of code and what it controls. In order to do this you must put a "style" attribute in the code of a line. This can allow for you to make one specific part of the code look prettier/more vibrant than other parts.

INTERNAL CSS

Internal css is a way to make a specific webpage look more unique and stylish. In order to do this you must put a "style" attribute in the head of the webpage.

EXTERNAL CSS

External css is a way to make a base template for all the needed css for a website. You must link the external css file by linking the stylesheet in the head for the external css to work.
