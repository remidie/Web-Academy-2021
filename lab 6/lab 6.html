Meet the Favicon

A favicon is a little image that is shown in the left side of a tab to help us differeniate them. Some popular examples are the youtube logo, the chrome logo that is shown when you open a new tab, and the colorful Google G logo. They are made by making an image and then inserting it into the webpage by using the code "link rel="shortcut icon" href="xxxx.png" type="image/x-icon"" but instead of "xxxx" you use the image file name and whichever folder they're in within the directory. From my experience this has been quite easy, with the hardest part being coloring in the logo and making/deciding on the design. It was quite tedious having to input the code for each individual pixel in a 32 x 32 image, but inserting it into the webpage was pretty easy and quick to do. I made the colored logo using binary code, with 9 bits per pixel.
